Brief detail about the dataset

This data is the historical stock market data for current S&P 500 companies, from 2014-2017. Each record represents a single day of trading, and includes the ticker name, volume, high, low, open and close prices.

This dataset was fetched from mavenanalytics website.

The research questions the analysis answered

  • Which date in the sample saw the largest overall trading volume? On that date, which two stocks were traded most?
  • On which day of the week does volume tend to be highest? Lowest?
  • On which date did Amazon (AMZN) see the most volatility, measured by the difference between the high and low price?
  • If you could go back in time and invest in one stock from 1/2/2014 - 12/29/2017, which would you choose?

Question 4

  • If you could go back in time and invest in one stock from 1/2/2014 - 12/29/2017, which would you choose?

To answer this question, we need to find the difference between the price for the dates. The code chunks below will do exactly that.

sp500_in_2014_and_2017 <- sp500 %>% 
  filter(date == '2014-1-2' | date == '2017-12-29') %>% arrange(symbol) %>%
  select(symbol, date, open, close)

This first code chunk will filter and group stocks for the two dates we are examining. After this, this next code will calculate and arrange the difference between the prices in order to determine the best performing stock between the selected period.

price_diff_2014_and_2017 <- sp500_in_2014_and_2017 %>% 
  group_by(symbol) %>% 
  mutate(price_diff = c(NA, diff(close))) %>%
  arrange(-price_diff) %>% 
  print(n = 10)
